# Git 指令 入门

个人使用GitHub 代码管理工具 和gitee

GitHub 是一个基于互联网的代码托管平台，可以帮助开发者存储、管理和协作开发代码。它是目前全球最大的开源社区，许多开源项目都托管在 GitHub 上。

# Github 简单的项目拉取更新

使用的Git工具

Git 有许多常用的命令，例如：

  1. git init：初始化一个新的 Git 仓库。
  2. git add：将文件或者文件夹添加到 Git 仓库的暂存区。
  3. git commit：将暂存区的文件提交到 Git 仓库。
  4. git clone：从远程仓库克隆一个本地版本。
  5. git pull：从远程仓库拉取最新的代码到本地。
  6. git push：将本地的代码推送到远程仓库。

# 利用 git 工具 克隆已有的项目

git@github.com:2002i/docs.git 是项目的 SSH 地址

运行命令，把项目复制克隆下了 然后进行操作

# 1‘克隆项目

命令： 克隆 git@github.com:2002i/docs.git

git clone git@github.com:2002i/docs.git

终端示例：

# 2’切换项目目录

把终端地址 切换到项目中 使用命令 cd [项目目录]

cd dist [dist 项目目录]

终端示例：
