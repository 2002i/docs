# 1panel 管理面板

1Panel 是新一代的 Linux 服务器运维管理面板 官网 https://1panel.cn/

# 安装1Panel

我这已ubuntu 为例：

安装环境

安装前请确保您的系统符合安装条件：

  • 操作系统：支持主流 Linux 发行版本（基于 Debian / RedHat，包括国产操作系统）；
  • 服务器架构：x86_64、aarch64、armv7l、ppc64le、s390x；
  • 内存要求：建议可用内存在 1GB 以上；
  • 浏览器要求：请使用 Chrome、FireFox、IE10+、Edge等现代浏览器；
  • 可访问互联网。

# 安装步骤：

  1. 终端运行 命令
  2. 设置 1Panel 安装目录（默认为/opt）
  3. 设置 1Panel 端口（默认为33023）
  4. 设置 1Panel 用户名称（默认为83df4dc7fe）
  5. 设置 1Panel 用户密码（默认为ceae6cd33b）

# 终端运行 命令

运行终端安装脚本

curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && sudo bash quick_start.sh
